HEX-RGB Converter

Converte o RGB fornecido para o valor HEX equivalente e vice-versa

The HEX code is used to represent colors on the web, especially in HTML and CSS. It's a compact, Hexadecimal-based way to define colors. The RGB format, on the other hand, use three numeric values from 0 to 255 to represent the amount of Red, Green and Blue. HEX is more compact, while RGB is easier to read.
